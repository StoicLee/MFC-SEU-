# MFC大作业-链表可视化

## 项目介绍：
本项目是基于MFC实现的一个简易链表结构可视化程序，能够增添节点，插入节点，删除节点，查找结点

程序类视图：
- CListNode(定义节点)
- CLinklist(定义链表)
- CLinklistCtrl(链表控制)
- CLinklistDraw(用于绘制链表)
## 编辑环境
vs2022
